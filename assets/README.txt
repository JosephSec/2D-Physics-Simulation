Use the Launch.bat file to launch the program

Input Controls:
	Esc: End Program
	
	Space: Pause/Unpause Time

	R: Reset Camera Position

	W/↑: Camera Up
	A/←: Camera Left
	S/↓: Camera Down
	D/→: Camera Right
	
	U: Burst Spawn (+1000 bodies)
	P: Move Black Hole To Cursor
	C: Clear Space

	Left Click:  Pull Objects
	Right Click: Push Objects

	Equals: Create New Body
	Minus:  Delete Previous Body

	Enter:     Create Static Body
	Backspace: Delete Static Body

	Scroll: World Zoom

UI Control Menu:
	~ + T: Toggle Menu Active
	~ + R: Reset Menu Position
  	~ + 1: Previous Tab
 	~ + 2: Next Tab
